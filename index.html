Amos

Amos is a movie database where you can create your favorite list of movies. You can browse between popular, upcoming, and now playing movies.

Amos is the final project for the General Assembly Javascript course class (js18) created by Samo Drole, and all the movie data on this page belongs to The Movie Database (TMDb).
